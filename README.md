# YouTube Rewind Controls for Firefox

![Extension Preview](icons/icon128.png)

A Firefox extension that adds 10-second rewind/forward buttons to YouTube player.

## Features

- Adds two large, visible buttons below YouTube player:
  - « 10s (rewind 10 seconds)
  - 10s » (forward 10 seconds)
- Buttons automatically hide when inactive (1 second after mouse leaves)
- Buttons reappear on mouse movement over player
- Keyboard shortcuts:
  - Left Arrow = rewind 10s
  - Right Arrow = forward 10s
- Works on all YouTube pages

## Installation

1. Download the extension files
2. Open Firefox and type `about:debugging` in the address bar
3. Click "This Firefox" in the left menu
4. Click "Load Temporary Add-on"
5. Select any file from the extension folder

## Building for Distribution

1. Zip all extension files (excluding development files)
2. Submit to [Mozilla Add-ons](https://addons.mozilla.org)
